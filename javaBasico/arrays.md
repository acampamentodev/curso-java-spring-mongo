## Utilizando Arrays

```java
int[] numerosDaSorte = new int[3];

numerosDaSorte[0] = 10;
numerosDaSorte[1] = 7;
numerosDaSorte[2] = 42;

System.out.println(numerosDaSorte[1]);
```

```java
double[] bimestre1 = { 7, 2.8, 9, 10, 8 };
double[] bimestre2 = { 6.5, 5, 7.5, 9.8, 7 };

System.out.println(bimestre1[3]);
System.out.println(bimestre1[0]);
System.out.println(bimestre2[1]);
System.out.println(bimestre2[4]);
```
