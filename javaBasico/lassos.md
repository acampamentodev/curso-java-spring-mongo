## Estruturas de repetição

### While

```java
int contador = 1;

while (contador < 5) {
  System.out.println(contador);
  contador++;
}
```

### Do While

```java
int contador = 1;

do {
  System.out.println(contador);
  contador++;
}
while (contador < 5);
```

### Fot

```java
for (int i = 1; i <= 10; i++) {
  System.out.println(i);
}
```

```java
for (int i = 10; i >= 1; i--) {
  System.out.println(i);
}
```
